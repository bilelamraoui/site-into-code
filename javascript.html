JavaScript

Introduction

Le JavaScript est un langage de programmation principalement utilisé dans les pages web intéractives, coté client, mais pouvant également être employé du coté serveur graçe au framework Node.js.

Le JavaScript s’écrit dans un éditeur de texte, et est interprété par les naviguateurs.

Syntaxe

En JavaScript, toute instruction doit se terminer par un point virgule ; 

            
              console.log("Bonjour monde!");

Tests conditionnels

If else

Le test If else renvoi au Si-Sinon en algorithmique, il permet de tester une expression booléenne et de donner des instructions a exécuter selon le resultat.

            
            //if est l’équivalent de « Si » en pseudo-code
            if (true) {

            }

            //Et else équivaut à « Sinon »
            else {

            }

Switch

le principe d’un Switch est équivalent au fonctionnement d’un Selon(cas) en algorithmique, il ne traite que des entiers, mais à l’avantage de les traiter au cas par cas, un bon exemple de cas pratique s’avère être le choix d’une langue par exemple :

            
              var francais = 0 ;
              var anglais = 1 ;
              var langue = francais ;

              switch (langue) { //Selon la langue
                case francais: //Dans le cas où langue = français ;
                	 console.log("Vous parlez français !");  //afficher un message indiquant la langue parlée.
                  break; //Fin selon
                case anglais :
            	  console.log("You speak english !");
                  break;
                 default: //Si ne parle ni français, ni anglais, instructions par défaut
                  console.log("Vous ne parlez aucune langue ?");
              }

Tables

Un tableau, ou un « array » comme on le dit en anglais, est comme son nom l’indique un tableau de valeur qui possède ses propres caractéristiques.

Un tableau crée verra ses valeurs se voir affecter un « indice » ou « index » allant de 0 au dernier element du tableau.

C’est graçe à cet index que l’on pourra récupérer l’élément qui nous interesse. Exemple : 
          
            var  tab = []; //Declaration du tableau tab
            tab.push(“elem”); //on affecte à tab la valeur “elem”
            console.log(tab[0]); // vaut “elem”

            var tab = [“rouge”, “bleu”, “vert”, “jaune”];
            tab.push(“violet”, “rose”);

            tab.lenght = 2; //réduit le tableau
            console.log(tab.lenght); // affiche la longueur du tableau

            object.keys(tab); // [‘0’, ‘1’, ‘2’, ‘3’]

Boucles

Le principe d’une boucle en JavaScript est de répéter une action tant que la condition de continuation est remplie, cependant, il faut bien faire en sorte que la condition de continuation soit fausse ou prenne fin à un moment donné, sinon, on rentrera dans une boucle infinie qui bouclera sans jamais executer le reste du code.

Les boucles permettent de gagner du temps en automatisant certaines tâches. Il existe différentes structures de boucles, mais ici nous ne verrons que le « for » Par exemple: 
            //On déclare et initialise un tableau contenant des couleurs sous forme de Chaines de caractère
          var tab = [“bleu”, “vert”, “jaune”,”rouge”];

          //On déclare une variable result pour stocker notre résultat
          var result; //Notre variable result

          //On execute notre boucle avec pour condition de continuation que la variable i, indice, soit inférieur à la longueur du tableau, et que i s’incrémente à chaque passage de la boucle.

          for (var i= 0 ; i < tab.lenght ; i++) {
                 result += tab[i];
          }

          //Donc notre variable result vaut result = result + tab[i]

Fonctions

Une fonction est un ensemble d’instruction regorupée dans le but d’exécuter une tâche préçise. Par exemple console.log() ; est une fonction qui affiche un message dans la console. L’interet de créer une fonction est de définir un outil pour une tâche préçise et d’y faire appel quand on en a besoin sans avoir à retaper l’ensemble du code. En JavaScript, il y a deux type de fonctions, les fonctions dites « natives » et celles que l’on crée soit même. Les fonctions natives sont intégrés de base dans le langage JavaScript, nul besoin de les créer, seulement de les « appeler ». Inutile de connaître le code qui compose console.log() ; pour l’appeler.

Par contre, si vous avez besoin de repeter une action, et qu’aucune fonction native n’existe dans JavaScript pour effectuer cette action, vous pouvez créer vous même cette fonction.
