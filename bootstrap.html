Bootstrap

Présentation

Bootstrap est un “framework” CSS développé dans la philosophie “mobile first” par Twitter Inc.

Bootstrap consiste en une série de feuilles de style, qui contiennent des classes CSS pré-établies, afin de faciliter le développement d’un site web en y insérant des éléments dynamiques ou permettant l’affichage sur tous les périphériques (tablettes, smartphones et bureau). Bootstrap permet donc d'implémenter le style de la mise en page directement sur le fichier html en intégrant les classes Bootstrap des balises. La documentation permet de trouver les classes dont vous avez besoin pour déployer votre site.

Composé de HTML, de CSS et de JavaScript, Bootstrap est actuellement le framework CSS le plus utilisé. La version actuelle de Bootsrap est v4.0.0-beta.2

On peut utiliser Bootstrap de deux façons: Soit par le biais d’un template à copier/coller dans votre éditeur de texte, que vous trouverez ci-dessous : 


          
          < html lang="en">
            < head>
              < title>Hello, world!< /title>
              < !-- Required meta tags -->
              < meta charset="utf-8">
              < meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">

              < !-- Bootstrap CSS -->
              < link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0-beta.2/css/bootstrap.min.css" integrity="sha384-PsH8R72JQ3SOdhVi3uxftmaW6Vc51MKb0q5P2rRUpPvrszuE4W1povHYgTpBfshb" crossorigin="anonymous">
            < /head>
            < body>
              < h1>Hello, world!< /h1>

              < !-- Optional JavaScript -->
              < !-- jQuery first, then Popper.js, then Bootstrap JS -->
              < script src="https://code.jquery.com/jquery-3.2.1.slim.min.js" integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" crossorigin="anonymous">< /script>
              < script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.3/umd/popper.min.js" integrity="sha384-vFJXuSJphROIrBnz7yo7oB41mKfc8JzQZiCq4NCceLEaO4IHwicKwpJf9c9IpFgh" crossorigin="anonymous">< /script>
              < script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0-beta.2/js/bootstrap.min.js" integrity="sha384-alpBpkh1PFOepccYVYDB4do5UnbKysX5WZXm3XxPqe5iKTfUKjNkCk9SaVuEZflJ" crossorigin="anonymous">< /script>
            < /body>
          < /html>

Soit en hébergeant vous même les classes Bootstrap avec votre site en les téléchargeant au préalable via la documentation officielle.

L’avantage de la seconde méthode est que si jamais Twitter modifie les classes de Bootstrap, votre site ne sera pas altéré à contrario de la methode du template qui prendra en compte les modifications des classes et pouvant perturber l’affichage de votre site.

La capacité d'adaptation à la taille de l'écran:

Bootstrap utilise le système de “breakpoint” afin de rendre responsive le contenu d'une page web. Les breakpoints sont basés sur les tailles minimums des écrans sur différents supports et ils permettent de mettre à l'échelle le contenu quand la taille de l'écran change. Bootstrap utilise 4 breakpoints de base, ceux-ci peuvent être modifiés par le développeur :

Nom Taille Notation
extra-small (téléphone en mode portrait) max-width: 575px xs
Small (téléphone en mode paysage) min-width: 576px, max-width: 767px sm
Medium (tablettes par exemple) min-width: 768px, max-width: 991px md
Large (écran d'ordinateur par exemple) min-width: 992px, max-width: 1199px lg
Extra-large min-width: 1200px xl

Par exemple :
.col-xs .col-sm .col-md .col-lg .col-xl

Le système de Grid:

Boostrap fonctionne avec un système de grid construit grâce à Flexbox. La page web est divisée en 12 colonnes ce qui permet une mise en page précise. Les différentes balises peuvent être pondérés pour prendre une largeur définie (de 1/12 à 12/12, la totalité de la largeur de l'écran).

Les éléments:

Bootstrap propose une panoplie d'élément déjà stylisés, dans un style moderne de flat-design. Cela permet d'insérer rapidement des éléments. Le style peut toujours être changé par le développeur. Dans la documentation la syntaxe et le rendu des éléments est expliqué en détails

Liste des éléments :
  • Alerte
  • Badge
  • Fil d'Arianne (breadcrumb)
  • Bouton
  • Groupe de boutons
  • Carte (card)
  • Diaporama (carousel)
  • Propriété collapse, pour cacher des éléments
  • Menu déroulant (dropdown)
  • Formulaire (form)
  • Champs de saisie (input group)
  • Mise en valeur d'un contenu spécial (jumbotron)
  • Liste (list group)
  • Boite de dialogue (modal)
  • Navigation (nav et navbar)
  • Paginatio, noter le numéro des pages
  • Fenêtre pop-up (popovers)
  • Progression (progress)
  • Indicateur de défilement (scrollspy)
  • Indicateur annexe (tooltips)

Les propriétés de mise en page:

Les classes Boostrap permettent de styliser les aspects des éléments. Ces classes se trouvent dans la partie Utilities de la documentation.
